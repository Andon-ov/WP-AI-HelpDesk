## 🎯 Описание

**Chatbot AI Engine** е мощен WordPress плъгин, който позволява на администраторите да интегрират AI-управляван чатбот на своя сайт. Плъгинът следва модела BYOK (Bring Your Own Key), което означава, че всеки потребител може да използва собствения AI API ключ.

### Ключови характеристики:

✅ **Абстрактна архитектура** - Не е обвързан с конкретна ниша или AI доставчик
✅ **BYOK модел** - Използвайте собствения AI API ключ
✅ **Множество доставчици** - OpenAI, Groq, Anthropic, Custom APIs
✅ **Лесна конфигурация** - Всички настройки в WordPress Admin панел
✅ **Минималистичен дизайн** - Модерен, отзивчив чат интерфейс
✅ **Висока безопасност** - Nonces, input validation, XSS protection
✅ **Локализация** - Подготвено за превод на други езици
✅ **Разширяемо** - Легко се добавят нови функции (RAG,876 и т.н.)

## 📦 Требования

### Минимални системни требования:

- **WordPress:** 5.0 или по-нова версия
- **PHP:** 7.4 или по-нова версия
- **MySQL:** 5.7 atau MariaDB 10.2+
- **SSL сертификат:** Препоръчвано за HTTPS комуникация

### Допълнителни изисквания:

- **AI API ключ** от един от поддържаните доставчици
- **Интернет връзка** за API комуникация

## 📥 Инсталация

Вижте **[INSTALLATION.md](./INSTALLATION.md)** за детални инструкции по инсталация.

**Бързо начало:**

1. Изтеглете плъгина
2. Качете го в `/wp-content/plugins/`
3. Активирайте го в WordPress Admin → Plugins
4. Отидете в **Администрация → AI Chatbot**
5. Конфигурирайте вашия API ключ

---

## ⚙️ Конфигурация

Вижте **[CONFIGURATION.md](./CONFIGURATION.md)** за пълна документация на всички настройки.

### Основни настройки:

```
1. Включване/Отключване на чатбота
2. Избор на AI доставчик
3. API URL (автоматично попълнен)
4. API Ключ
5. Име на модел (напр. gpt-3.5-turbo)
6. Системен промпт
7. Максимални токени
8. Температура (креативност)
9. Позиция на виджета
```

## 🚀 Употреба

### За крайни потребители:

1. **Кликнете на мехурчето** - Плаващия чат интерфейс в долния десен ъгъл
2. **Напишете вашето съобщение** - Входното поле е готово
3. **Натиснете Enter или кликнете Send** - Съобщението се изпраща
4. **Получете отговор от AI** - Чакайте за отговора

### За администраторите:

1. Отидете в **Администрация → AI Chatbot**
2. Заредете вашия AI ключ
3. Конфигурирайте поведението на чатбота
4. Нагласете системния промпт според вашите нужди
5. Включете чатбота

## 🤖 Поддържани AI доставчици

### 1. **OpenAI** (По подразбиране)

- **API URL:** `https://api.openai.com/v1/chat/completions`
- **Модели:** gpt-3.5-turbo, gpt-4, gpt-4-turbo и др.
- **Документация:** https://platform.openai.com/docs

### 2. **Groq**

- **API URL:** `https://api.groq.com/openai/v1/chat/completions`
- **Модели:** mixtral-8x7b-32768, llama2-70b-4096 и др.
- **Документация:** https://console.groq.com/docs

### 3. **Anthropic (Claude)**

- **API URL:** `https://api.anthropic.com/v1/messages`
- **Модели:** claude-3-opus, claude-3-sonnet и др.
- **Документация:** https://docs.anthropic.com

### 4. **Персонализирани API**

- **Поддръжка за всяка OpenAI-съвместима API**
- **Напълно конфигурируемо URL и заглавие**

## 🔒 Безопасност

### Реализирани мерки за безопасност:

1. **Nonce верификация** - Всички AJAX заявки са защитени
2. **Input валидация** - Всички входни данни се проверяват
3. **Санитизация** - Специални функции за очистка на данни
4. **XSS защита** - Екранирани всички потребителски входи
5. **CSRF защита** - WordPress нонс система
6. **Пароли в администрацията** - API ключ се съхранява безопасно
7. **SSL/HTTPS** - Препоръчено за цялата комуникация
8. **権限проверка** - Само администраторите могат да конфигурират
